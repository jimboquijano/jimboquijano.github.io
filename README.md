# Jimbo Quijano Portfolio

**Website:** [https://jimboquijano.github.io](https://jimboquijano.github.io)

A single-page portfolio website showcasing my work, projects, and skills. Built in a weekend as a personal project to demonstrate frontend development skills, clean design, and interactive features.

---

## Features

- **Single-page design:** Smooth scrolling navigation for seamless user experience.
- **Responsive layout:** Optimized for desktop, tablet, and mobile devices.
- **Interactive starfield background:** Dynamic 3D starfield effect for visual engagement.
- **Modal contact form:** Easy way for visitors to reach out with smooth open/close animations.
- **ScrollSpy navigation:** Highlights the current section in view as the user scrolls.
- **Dynamic link wrappers:** Clickable cards and project items with external links.
- **Responsive nav and footer:** Adjusts layout based on viewport width.

---

## Tech Stack

- **HTML5 & CSS3 / SCSS** – Modern, responsive layout styling.
- **JavaScript (ES6 modules)** – Interactive components and animations.
- **Vanilla JS libraries** – Custom-built ScrollSpy, Starfield, ResponsiveNav, and Modal classes.
- **GitHub Pages** – Hosting the live website.
